# Monika
Mnemonic phrase generator. Can be used in cases that requring a ballance between difficulty and safety.

#How to use
1)Launch the program.
2)Choose the "Generate keys" option to test the program or choose "Settings" to set it up.
3)Set amount of key files to be generated, amount of words to be included and the folder for results in "Settings".
4)Run "Generate keys".
5)Get results in "results" or specified folder.

Make sure that the program has "wordlist.txt" file in the same folder before its start or the wordlist is not empty.

# Monika
Генератор мнемонических фраз. Может использоваться в случаях, когда требуется баланс между сложностью и безопасностью.

#Как пользоваться
1)Запустите программу.
2)Выберите опцию "Cгенерировать ключи", чтобы протестировать программу, или выберите "Настройки", чтобы настроить её.
3)Укажите количество файлов ключей, которые будут сгенерированы, количество слов, которые будут включены, и папку для результатов в разделе "Настройки".
4)Запустите "Сгенерировать ключи".
5)Получите результаты в "результатах" или указанной папке.

Перед запуском программы убедитесь, что в той же папке находится файл "wordlist.txt", или что список слов не пуст.
